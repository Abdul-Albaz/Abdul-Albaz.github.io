Set up Subscribe Form

To set up the subscribe form to send emails with the persons which are contacting you, open "subscribe.php" file and change the email from line 26 with your own email. 

Line 26: $address = "example@example.net";

 

Set up Contact Form

To set up the contact form to send emails with the persons which are contacting you, open "contact.php" file and change the email from line 42 with your own email. 

Line 42: $address = "example@example.net";

 

Set up Google map

Copy the sources embed code from https://maps.google.com/ and replace the code with this existing code 

 <iframe src="https://maps.google.com/?ie=UTF8&ll=-37.817682,144.957595&spn=0.01134,0.026157&t=m&z=16&output=embed" width="100%" height="240" frameborder="0" style="border:0"></iframe>

 

Update Coming soon Countdown

To change the date of the countdown, go to "js" folder and open "custom.js" file. Change the date 

var austDay = new Date("December 27, 2014 00:00:00");
